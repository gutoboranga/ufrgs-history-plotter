Gráfico do Histórico Escolar UFRGS


Projeto que gera um gráfico do histórico escolar de alunos da UFRGS apresentando as notas ao longo do tempo (semestres).

É usado como entrada um arquivo html que pode ser obtido no site da UFRGS.


Instruções:

1. Para obter o arquivo html, faça login no site da UFRGS e acesse:

https://www1.ufrgs.br/intranet/portal/public/index.php?cods=1,1,2,4

2. Salve a página em html (Arquivo > Salvar Página como ou Ctrl + S)

3. Use o botão abaixo para carregar o arquivo e aguarde uns instantes.
